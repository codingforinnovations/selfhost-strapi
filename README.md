### ðŸ† Getting Started:

- Clone Repo

```bash
cd ~/Dev
mkdir ~/Dev/selfhost-strapi -p
cd ~/Dev/selfhost-strapi
git clone https://github.com/codingforinnovations/selfhost-strapi.git .
```
- Copy `.env.example` -> `.env`

```bash
cp .env.example .env
```

- Run strapi!

```bash
docker compose up --build
```
